读在2017

12-31-2017

由于一些原因，这一年比往年多读了不少书，而且基本是集中在岁末三个月。除了基本的知识摄取和休闲娱乐外，另一个目的是想从中寻找一些问题的答案，而这些问题让我很是恼火。虽最终未能如愿，但或多或少影响了认知，“不求甚解”的一个好处可能就在于潜移默化吧。

总体来说，这一年读书比较杂。究其原因，一方面是由于自身所处困境的实际需求，另一方面是由于上大学以后阅读量急剧减少(惭愧…)，欠的阅读债比较多，再者便是专业上的需求。按学科划分来讲，社会科学方面主要集中在心理学，人文方面主要是文学和文学史，自然科学方面则是本行计算机（就不放在这里了）。这里主要挑出一些对我有些影响的书。

心理学

心理学方面，主要是从入门读物、基础教材、细分流派和研究领域几个方面来阅读的。整理了一个豆列。这里挑了我看过的和比较感兴趣的方向。

少有人走的路 ★★★★☆8.4 心智成熟的旅程 / [美] M·斯科特·派克 / 吉林文史出版社

第一本，非它莫属。开篇第一句便透露了生活的真相。后面自律、爱、成长和信仰等几部分更是颇有同感。

我们时代的神经症人格 ★★★★★9.0 卡伦·霍尼 / 译林出版社

本身作为患者，霍尼关于神经症的分析淋漓尽致。

活出生命的意义 ★★★★★8.7 [奥] 维克多·弗兰克 / 华夏出版社

可能，你有那么一段时间，“一切对我来说都毫无意义，突然之间”（《我是个年轻人，我心情不太好》），可以看看这本书。事业、爱与体验、苦难本身的意义。

改变心理学的40项研究 ★★★★★8.8 探索心理学研究的历史 / 〔美〕Roger R Hock著 / 中国轻工业出版社

心理学的一些著名研究成果。

这才是心理学（第9版） ★★★★★9.0 看穿世界的批判性思维 / 基思·斯坦诺维奇 (Keith E.Stanovich) / 人民邮电出版社

用科学讲心理学。（其实不如说是用心理学讲科学……）

心理学导论 ★★★★★9.3 思想与行为的认识之路 / Dennis Coon / 中国轻工业出版社

不错的心理学入门教材，通俗易懂，内容丰富。还有《津巴多普通心理学》和《心理学与生活》这两本书也不错，可惜图书馆一直没借到，也就还没看。

人格心理学 ★★★★★8.9 万千心理 / Jerry M. Burger / 中国轻工业出版社

各流派关于人格的理论。

发展心理学 ★★★★★8.7 人的毕生发展 / [美] 罗伯特·费尔德曼 / 世界图书出版公司北京公司

关于发展心理学的教科书， 重点看了青春期(12~20)、成年早期(20~40)、成年中期(40~65)三部分 ，很不错。

登天的感觉 ★★★★☆7.8 我在哈佛大学做心理咨询 / 岳晓东 / 上海人民出版社

这本书对简单了解心理咨询和治疗很有用，通俗易懂，案例和结构安排很合理。

文学

步履不停 ★★★★★8.7 [日] 是枝裕和 / 北京联合出版公司

“人生，总有那么一点来不及”，但是，也总有那么多来得及。

过于喧嚣的孤独 ★★★★★8.8 (捷克) 赫拉巴尔 / 北京十月文艺出版社

在孤独与人文中“胡思乱想”到极致。

写作这回事 ★★★★★8.6 创作生涯回忆录 / [美]斯蒂芬·金 / 上海译文出版社

作家的成长之路，写作的技巧和感悟。

百年孤独 ★★★★★9.2 [哥伦比亚] 加西亚·马尔克斯 / 南海出版公司

无需多言，魔幻现实主义巨著。

人间失格 ★★★★☆8.3 太宰治 / 作家出版社

你还能比太宰治颓？

我是个年轻人 我心情不太好 ★★★★☆7.5 (挪威)阿澜·卢 / 浙江文艺出版社

我想，这本书比任何人都懂此时的我，包括我自己。

我已经结婚了 我心情还不好 ★★★★☆7.6 [挪威]阿澜·卢 / 浙江文艺出版社

不及《我是个年轻人，我心情不太好》，不过北欧当代小说的笔触真是很有意思的。

亲爱的安德烈 ★★★★★8.7 龙应台 / 人民文学出版社

“ 什么样的工作比较可能给你快乐？第一，它给你意义；第二，它给你时间。当你的工作在你心目中有意义，你就有成就感。当你的工作给你时间，不剥夺你的生活，你就有尊严。成就感和尊严，给你快乐。 ”

小王子 ★★★★★9.0 [法] 圣埃克苏佩里 / 人民文学出版社

小王子不小，他会成长，成长为生活的巨人。

我们仨 ★★★★★9.0 杨绛 / 生活·读书·新知三联书店

“送一程，说一声再见，又能见到一面。离别拉得长，是增加痛苦还是减少痛苦呢？我算不清。但是我陪他走得愈远，愈怕从此不见。”

许三观卖血记 ★★★★★8.9 余华 / 作家出版社

算是中国为数不多的优秀现实主义作品吧，余华的小说都很针砭时代。

沉默的大多数 ★★★★★9.3 王小波 / 湖南文艺出版社

王小波的文字真是太有意思，那股带戏谑讽刺的聪明劲和幽默感跃然纸上。

月亮和六便士 ★★★★★9.0 [英] 毛姆 / 上海译文出版社

梦想与现实。

白夜行 ★★★★★9.1 [日] 东野圭吾 / 南海出版公司

故事本身、叙述和描写真的是极其精彩。

牧羊少年奇幻之旅 ★★★★★8.6 [巴西]保罗·柯艾略 / 南海出版公司

你要走多远才回望起点？

一个陌生女人的来信 ★★★★★8.9 [奥] 茨威格 / 人民文学出版社

果然意识流大师，写得我都想谈恋爱了。

生活在别处 ★★★★☆8.4 [捷克] 米兰·昆德拉 / 上海译文出版社

看看诗人的一生。

世界文学史 ★★★★☆7.9 张德明 / 浙江大学

外国文学史入门。

欧美文学导读 ☆☆☆☆☆评价人数不足 卢锦明 / 北京大学出版社

欧美文学史入门不错。

这一年的阅读很杂，但是也逐渐露出轮廓。明年打算系统地阅读下现实主义的文学作品（目前感觉现实主义的目标和手法和我的兴趣更一致些），北欧的小说也多关注一下。然后，就是边看着，边想着了。
